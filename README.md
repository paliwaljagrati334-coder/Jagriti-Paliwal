# 📦 Supply Chain Dashboard | Power BI Project

An interactive **Supply Chain Analytics Dashboard** built in **Microsoft Power BI** to monitor key business metrics such as revenue, profit, sales performance, regional trends, order status, and cost efficiency.

This dashboard helps businesses make data-driven decisions by providing real-time insights into operations and performance.

---

## 🚀 Project Overview

The main objective of this project is to analyze supply chain operations and visualize important KPIs for better strategic planning.

It enables stakeholders to:

- Track Total Revenue, Profit & Sales
- Monitor Operational Costs
- Analyze Regional Performance
- Understand Product Category Trends
- Review Order Status Distribution
- Compare City / Country Level Performance
- Identify Growth Opportunities

---

### 🔹 KPI Cards
